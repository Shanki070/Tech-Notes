Basic Syntax

Package Definition

In My Language, every source file must begin with a package definition. This helps in organizing your code into namespaces.

package com.example.mypackage

Packages can be nested. Here is an example:

package com.example.mypackage.subpackage

Imports

To use functions or classes from other packages, you need to import them.

import com.example.mypackage.SomeClass

You can also import all classes from a package:

import com.example.mypackage.*

Basic Imports

This section covers basic import statements.

import com.example.basic.BasicClass

Advanced Imports

This section covers advanced import techniques.

import com.example.advanced.AdvancedClass

Basic Types

My Language has several basic types. Here are a few examples:
